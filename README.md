# ReactKTraining

React 기초 학습 과정에서 만든 연습용 저장소입니다. 컴포넌트 분리, 기본 상태 관리, 화면 구조를 익히기 위해 사용했습니다.

## Status

- React 학습 기록을 보관한 아카이브입니다.
- 운영 서비스나 대표 포트폴리오가 아니라, 초기 학습 과정을 남긴 저장소입니다.
- 현재 지원용 대표 포트폴리오는 `ketose-portfolio`에서 관리합니다.

## 학습 범위

- React 컴포넌트 분리
- 기본 상태 관리
- CSS 기반 화면 구성
- 목록, 게시글, 푸터 등 화면 단위 구성 실습

## 실행

```bash
npm install
npm start
```
